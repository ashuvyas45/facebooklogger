Functions

  • logs = viablimp.token('TOKEN')
    • This returns a viablimp object. The TOKEN should be obtained from facebook chatbot.
  • logs.message(MESSAGE)
    • This returns response status code. 200 for OK. 404 for not found. See response status codes for more info.
    If the token is incorrect or in case of no network, it raises error.
    Here MESSAGE can be anything to log. String, array, dictionary, etc.
  • logs.add_name('SERVICE_NAME')
    • Returns True. All log messages sent by calling from the viablimp object now will have the SERVICE_NAME as their heading as illustrated below.
